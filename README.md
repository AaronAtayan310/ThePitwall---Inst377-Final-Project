# ThePitwall

ThePitwall is a Formula 1 data dashboard that gives casual fans live and up-to-date race weekend information in a clean, easy-to-navigate interface. Whether you missed a qualifying session, want to follow live lap times during a race, or just want to explore pit stop strategies and car telemetry, ThePitwall is the central command center to receive any and all F1 info.
The site pulls real F1 data from the [OpenF1 API](https://openf1.org/) and is built with Node/Express for the backend and JavaScript for the frontend.

## Target Browsers

ThePitwall is designed as a **desktop-first web application** and is tested on the following browsers:

- **Google Chrome** (v120+) — primary target, recommended
- **Microsoft Edge** (v120+)
  Mobile browsers (iOS Safari, Android Chrome) are **not officially supported** at this time.
